## dev1

（命名dev为develop版本）

- 2026.4.16更新

1. 新建文件夹（这其实是在4.15做的）
2. 硬编码了5x5地图，2个阵营，10个棋子
3. 添加了piece_config.js文件用于记录棋子的信息，目前只有id，name，color三个属性
4. 游戏中棋子有三种移动方式，但目前共有一种逻辑，就是直接移动
5. 游戏中棋子有三种投掷物，分别为雷，烟，火，实现了基础逻辑
6. 每个棋子都有100血和5精力，但后续可能会使用js文件更改其基本属性

## dev2

- 2026.4.17更新

1. 更新回合系统：回合将定义为游戏内胜负结算的时间单位，玩家规划和执行棋子逻辑的时间单位为行动数，在顶部UI显示中以以下格式命名

```
回合 1：行动 1
```

2. 修复烟雾弹逻辑：烟雾将阻挡烟雾中的棋子视线，烟雾作为终点时不阻挡视线
3. 添加棋子死亡：棋子血量为0及以下时，移除棋子并使其卡片变灰
4. 修复棋子执行逻辑bug，当前棋子执行逻辑遵循以下基本原则：

```
1. 不同阵营的同一执行顺序的棋子同时执行
2. 同一棋子所有操作同时执行（特殊地，实现了棋子可以被自己投出的投掷物砸到）
3. 棋子死亡时，所有在该棋子后执行的同一阵营棋子的执行顺序提前
```

## dev3

- 2026.4.18更新

1. 新增射击判定系统，所有棋子依据反应值依次行动，行动后立刻向视野内的随机敌方单位开枪，目前默认造成10点伤害
2. 新增卡片附属浮动面板及其视觉效果，该浮动面板将记录棋子的枪械和投掷物，目前仅做了枪械，且没有实际效果
3. 新增了glock和usp-s两种枪械
4. 修复bug

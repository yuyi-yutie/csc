可以通过访问下方链接游玩


## dev1

（命名dev为develop版本）

2026.4.16更新

①新建文件夹（这其实是在4.15做的）

②硬编码了5x5地图，2个阵营，10个棋子

③添加了piece_config.js文件用于记录棋子的信息，目前只有id，name，color三个属性

## dev2

2026.4.17更新

①更新回合系统：回合将定义为游戏内胜负结算的时间单位，玩家规划和执行棋子逻辑的时间单位为行动数，在顶部UI显示中以以下格式命名

```
回合 1：行动 1
```

②修复烟雾弹逻辑：烟雾将阻挡烟雾中的棋子视线，烟雾作为终点时不阻挡视线

③添加棋子死亡：棋子血量为0及以下时，移除棋子并使其卡片变灰

④修复棋子执行逻辑bug，当前棋子执行逻辑遵循以下基本原则：

```
1. 不同阵营的同一执行顺序的棋子同时执行
2. 同一棋子所有操作同时执行（特殊地，实现了棋子可以被自己投出的投掷物砸到）
3. 棋子死亡时，所有在该棋子后执行的同一阵营棋子的执行顺序提前
```
